# index-28-guide

## 项目简介

index-28-guide 是一个用于归档和展示独立 HTML 页面的通用仓库。这里存放着多个功能独立的静态页面，适用于学习参考、快速原型演示或小型工具页面存档。每个页面均为独立完整的 HTML 文件，不依赖外部框架或复杂后端环境。

## 目录说明

```
index-28-guide/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面共用的静态资源（CSS、JS、图片等）
├── README.md       # 本文件
└── LICENSE         # 许可证文件
```

- **pages/**：每个子文件夹或文件对应一个独立页面，命名清晰，便于识别。
- **assets/**：存放页面之间可能共用的样式、脚本或媒体资源，避免重复。

## 页面归档说明

本仓库中的页面均为静态 HTML 文件，可直接在浏览器中打开使用。它们可能涵盖以下类型：

- 实用工具页面
- 交互式演示
- 信息展示页
- 功能小部件

每个页面内部均包含完整的结构与样式，无需额外配置即可运行。部分页面可能引用 assets 目录下的资源，请保持相对路径正确。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 打开 `pages/` 目录，找到感兴趣的 HTML 文件。
3. 双击或在浏览器中直接打开即可查看效果。

若希望将页面部署到静态托管服务，可直接上传 `pages/` 目录内容（或整个仓库），并确保资源路径正确。
